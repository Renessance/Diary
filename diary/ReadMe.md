Реализованы namespace и autoloading.

Проект реализован по модели MVC в проекте есть базовые классы в папке core отвечающие за вид, модель, контроллер и маршрутизацию. 

Маршруты сделаны статичными поскольку проект не большой.

Реализован Access List Controller чтобы ограничить доступ на страницы не принадлежащие пользователям.
